# ExpoMessaging - React Native 聊天应用

这是一个使用 Stream Chat SDK 构建的 React Native 聊天应用示例。

## 🚀 快速开始

### 在手机上查看
1. 克隆项目：`git clone https://github.com/pc236r/exui-react-native.git`
2. 进入目录：`cd exui-react-native/examples/ExpoMessaging`
3. 安装依赖：`yarn install`
4. 启动服务器：`npx expo start`
5. 在手机上安装 Expo Go 应用
6. 扫描终端中的二维码

## 📱 项目特性
- React Native + Expo
- Stream Chat SDK 集成
- Expo Router 导航
- 支持 iOS 和 Android
